# KISS

Usage: ./kiss.sh [OPTIONS]...

Description:
  A simple local Linux enumeration script for beginners.
  Enable info boxes to see what each module does and get tips on what to look for.

Options:
*  --nocolor          Disable colors
*  -i, --info         Enable info boxes
*  -e, --export PATH  Enter a path for export
*  -d, --debug        Debug modus
*  -t, --tough        Enable tough mode (more checks)
*  -h, --help         Displays this help text
